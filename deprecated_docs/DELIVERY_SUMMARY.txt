 IMPORTANT NOTES

For Machine Learning:
   ALWAYS use WAV format (lossless)
   MP3 introduces compression artifacts
   Metadata contains all ground truth labels
   Filename convention includes parameters

For Supercomputer Deployment:
   Consider multiprocessing for speed
   Use fast SSD storage
   Split into smaller batches if >5000 clips
   Monitor disk space (~10 MB per clip)

For Best Results:
   Upload high-quality vehicle sounds
   Start with small batches (10-100) to test
   Review metadata for accuracy
   Check generation_log.txt for errors
